About This Project

Project Goal

The Football Scout Tool enables data-driven discovery and analysis of professional football (soccer) players through interactive visualizations. The goal is to help users understand player performance beyond traditional statistics by visualizing advanced metrics like expected goals (xG), progressive carries, and percentile rankings across Europe's top 5 leagues.

Intended Audience

This tool is designed for:

  • Football fans who want to understand player performance beyond goals and assists
  • Amateur analysts looking to compare players across different leagues and positions
  • Data enthusiasts interested in exploring advanced football metrics
  • Students studying sports analytics and data visualization

Key Features

Users can compare players side-by-side, explore interactive scatter plots with dynamic filtering, find statistically similar players using machine learning, and analyze position-specific performance metrics across 2,000+ players from the 2024-2025 season.

Data Source

Dataset: Football Players Stats 2024-2025 (originally sourced from FBref via Kaggle)
Coverage: 2,000+ players across Premier League, La Liga, Bundesliga, Serie A, and Ligue 1
Metrics: 250+ statistics including goals, assists, expected goals (xG), progressive carries, tackles, passing accuracy, and more

Technology

Built using Altair (Vega-Lite) for interactive visualizations, Python with Pandas for data processing, and scikit-learn for statistical similarity analysis. All visualizations are fully interactive with tooltips, filtering, and export capabilities.
